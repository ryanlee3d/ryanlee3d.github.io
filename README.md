# Ryan Lee Portfolio

A personal portfolio website for Ryan Lee, a full-stack developer and recent Computer Science graduate. The site highlights selected projects, technical skills, professional focus areas, and contact links.

## Overview

This portfolio is built as a React single-page application using Vite. It includes sections for featured projects, skills, about information, and contact details, with responsive navigation for desktop and mobile visitors.

## Featured Work

- **Festa Italia Website** - A deployed full-stack application built with React, Supabase, PostgreSQL, authentication, and team-based Jira/SCRUM workflows.
- **Behind the Opioid Epidemic** - An interactive data visualization project using Python, Pandas, JavaScript, D3.js, and ObservableHQ.
- **3D First Person Shooter Game Development** - A Java/OpenGL game architecture project using Blender assets and the TAGE game engine.

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

Run linting:

```bash
npm run lint
```

## Project Structure

```text
src/
  assets/      Project images and static assets
  App.jsx      Main portfolio content and layout
  App.css      Portfolio styling
  main.jsx     React entry point

```
